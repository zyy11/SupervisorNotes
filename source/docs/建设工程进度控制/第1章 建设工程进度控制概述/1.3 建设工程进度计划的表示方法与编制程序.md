# 1.3 建设工程进度计划的表示方法与编制程序

## 进度计划表示方法

- 横道图（甘特图）
- 网络图

## 进度计划编制程序
- **计划准备阶段**
    * 调查研究
    * 确定进度计划目标
        - 时间目标
        - 时间-资源目标
        - 时间-成本目标
- **绘制网络图**
    * 项目分解（前提）
    * 分析逻辑关系
        - 主要依据方案、资源、经验
    * 绘制网络图
- **确定时间参数**
    * 计算工作持续时间
        - 时间定额、产量定额
    * 计算网络计划时间参数
    * 确定关键线路和关键工作
- **网络计划优化**
